# Dejavu Bot

Telegram-бот для учёта просмотренных фильмов по ссылкам из чата киноклуба. Реагирует на ссылки, начинающиеся словом "смотрим". Бот проверяет ссылку на повторы по базе, идет по ссылке, ищет название фильма и проверяет фильм на повторы по базе, но уже по названию. По результатам проверки бот может удалить сслыку с сообщением "Этот фильм уже смотрели" (нужно дать соответствующие права боту при добавлении в чат), либо оставить ссылку в чате с добавлением нового фильма в базу.

## Технологии
- Python 3.11+
- aiogram 3
- sqlite3
- aiohttp + BeautifulSoup

## Быстрый старт
1. Создайте виртуальное окружение и установите зависимости:
   ```bash
   python -m venv .venv
   .venv\\Scripts\\activate
   pip install -r requirements.txt
   ```
2. Создайте `.env` на основе `.env.example` и укажите `BOT_TOKEN`.
3. Запустите бота:
   ```bash
   python -m bot.main
   ```

## Переменные окружения
- `BOT_TOKEN` (обязательно): токен Telegram-бота.
- `ALLOWED_CHATS` (необязательно): список ID чатов через запятую, где бот разрешён.

Пример:
```env
BOT_TOKEN=123456:ABCDEF...
ALLOWED_CHATS=-1001234567890,-987654321
```

## Деплой
Для деплоя в Amvera используется файл `amvera.yml`.
Команда запуска: `python -m bot.main`.

## Безопасность
- Не публикуйте `.env`, токены, пароли и рабочие дампы базы данных.
- Если токен уже попадал в открытые файлы или логи, перевыпустите его через BotFather.
